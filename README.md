## robot-helper

This is a chatbot based on baidu voice synthesis and Turing robot

![robot-helper](https://raw.githubusercontent.com/Yangfan2016/PicBed/master/Blog/robot-helper.gif)

### Run

```bash
$ yarn install
$ yarn run dev
```
